# Lorenzo Active Hub — Sales Funnels (MECE)

## ICP (Ideal Customer Profile)

### Segmento A: Turistas en Lima
- **Perfil**: Visitantes internacionales (20-45), hospedados en Miraflores/Barranco, buscando actividades fuera del circuito estandar
- **Ventana de reserva**: 0-48 horas (espontaneo)
- **Idioma**: Ingles primero, algo de español
- **Gasto**: Alta disposicion por sesion, baja repeticion
- **Descubrimiento**: Google Maps, TripAdvisor, concierge de hotel, Airbnb Experiences, blogs de viaje, Instagram explore
- **Driver de conversion**: Novedad + autenticidad + facilidad de reserva
- **Insight clave**: Buscan exactamente lo que Lorenzo ofrece — autentico, local, no montado para turistas. La Punta es "the place the guidebook didn't mention"

### Segmento B: Locales Selectivos
- **Perfil**: Residentes de Lima/Callao (25-40), clase media-alta, buscando escape de fin de semana, orientados a deporte/salud o curiosidad cultural
- **Ventana de reserva**: 3-7 dias (actividad planificada de fin de semana)
- **Idioma**: Español primero
- **Gasto**: Moderado por sesion, alto potencial de repeticion
- **Descubrimiento**: Instagram, referencia de amigos/familia, Google Maps, prensa local
- **Driver de conversion**: Narrativa de escape + comunidad + acceso recurrente
- **Insight clave**: Son los que salen de la inercia de Miraflores/Barranco. "In the know". Una vez que van, vuelven y traen amigos. La Costanera hace que 15-20 minutos sea viable para un plan de sabado

---

## Embudo Universal

Todos los funnels terminan igual:

```
Descubrimiento → lorenzo.club o perfil de Instagram → Conversacion WhatsApp →
Respuesta humana → Sesion reservada → Experiencia en sitio →
Boca a boca (el ciclo reinicia)
```

La conversacion de WhatsApp ES el proceso de venta. No hay checkout, no hay pasarela de pago, no hay widget de reservas. Esto es intencional — preserva la interaccion humano-a-humano que define la marca. La restriccion (sin booking online) es en realidad la feature.

### Categoria 1: DESCUBRIMIENTO ORGANICO
*Sin gasto pagado — atencion ganada*

#### 1A. Instagram Organico
| | |

| **Segmento** | A + B |

### Categoria 2: PARTNERSHIPS
*Apalancando distribucion existente*

#### 2A. Red de Concierges de Hoteles/Hostales
| | |
|---|---|
| **Segmento** | A (turistas) |
| **Mecanica** | Concierge recomienda Lorenzo → huesped escribe por WhatsApp |
| **Prioridad** | P1 |
| **Esfuerzo** | Medio (construccion de relaciones, presencia fisica) |
| **Conversion estimada** | 15-25% de referidos del concierge reservan |

**Tacticas:**
- Tarjetas impresas (estetica de marca, no flyers) en hoteles boutique de Miraflores/Barranco
- Relacion personal con staff de concierge (no partnerships corporativos)
- Comision por sesion reservada (informal)
- Foco: hoteles boutique y hostales (no Marriott — audiencia equivocada)

#### 2B. Red de Hosts de Airbnb
| | |
|---|---|
| **Segmento** | A (turistas) |
| **Mecanica** | Host recomienda Lorenzo en su guidebook o mensajes → huesped contacta |
| **Prioridad** | P2 |
| **Esfuerzo** | Medio (construccion de relaciones) |
| **Conversion estimada** | 10-15% de huespedes referidos reservan |

**Tacticas:**
- Identificar top hosts en Miraflores/Barranco/Callao
- Sesion cortesia para que el host viva la experiencia primero
- Quick-link de WhatsApp para que compartan con huespedes
- Framing de "recomendacion local" (no publicidad)

#### 2C. Cross-Promotion con Negocios Locales
| | |
|---|---|
| **Segmento** | B (locales) |
| **Mecanica** | Cafes/restaurantes/tiendas locales muestran presencia de Lorenzo → curiosidad → visita |
| **Prioridad** | P3 |
| **Esfuerzo** | Bajo (organico, basado en relaciones) |
| **Conversion estimada** | Bajo individualmente, acumulativo en el tiempo |

**Tacticas:**
- Negocios de Callao/La Punta (no cadenas limenas)
- Material impreso coherente con marca (estilo zine, no brochure)
- Promocion reciproca: Lorenzo recomienda la cevicheria, la cevicheria recomienda Lorenzo

### Categoria 3: ADQUISICION PAGADA
*Gasto pagado para alcance segmentado*

#### 3A. Instagram/Meta Ads
| | |
|---|---|
| **Segmento** | A + B (geo-targeting diferenciado) |
| **Mecanica** | Turista: GPS Miraflores/Barranco + interes en deportes acuaticos/viaje → web → WhatsApp. Local: Lima metro + interes en actividades de fin de semana → web → WhatsApp |
| **Prioridad** | P2 |
| **Esfuerzo** | Medio (produccion creativa, testing de audiencias) |
| **CPA objetivo** | <$3 por conversacion, <$15 por sesion reservada |

**Enfoque creativo:**
- Video documental (15s max), NO graficas promocionales
- Estetica B&W coherente con marca
- Copy directo y minimo: "Paddle en La Punta. Reserva por WhatsApp."
- CTA "Enviar mensaje" hacia WhatsApp — NO "Compra ahora"
- Budget inicial: $5-10/dia, testear 3 audiencias, escalar ganadores

#### 3B. Google Ads (Search)
| | |
|---|---|
| **Segmento** | A (turistas, alta intencion) |
| **Mecanica** | "paddle boarding Lima" / "things to do La Punta" → ad → lorenzo.club → WhatsApp |
| **Prioridad** | P3 |
| **Esfuerzo** | Bajo (set and maintain) |
| **CPA objetivo** | <$5 por conversacion |

**Enfoque:**
- Solo Search (no Display — Display es off-brand)
- Exact/phrase match en queries de alta intencion
- Landing page: /en para queries en ingles, /es para español
- Budget: $3-5/dia, solo long-tail keywords

## Anti-Patrones (Lo Que Lorenzo NO Debe Hacer)

- Campañas de email masivo
- Pop-ups de descuento en el sitio web
- Tacticas de urgencia ("oferta limitada", "ultimos cupos")
- Partnerships con influencers lifestyle que no coinciden con el tono documental
- Marketing en grupos de Facebook
- Groupon / agregadores de descuento
- Chatbot automatizado reemplazando la respuesta humana en WhatsApp
- Anuncios en Google Display Network
- Posts de redes sociales con fotos stock genericas
- Programas de puntos/lealtad (se siente corporativo)
- Descuentos por volumen que atraigan al publico equivocado
